- About Section: Display information about Frankie, Kevin Gill's connection, and the meme potential.
- Token Details: Show the token supply (1 billion) and contract address (F8dgv9rSGQJQFUjNktHeGKsJjvfmn7HHutZLoApqpump).

## Style Guidelines:

- Primary color: Vibrant sky blue (#7DF9FF) to invoke dog collars and playful associations.
- Background color: Light grayish-blue (#D3E4E6), keeping with the light and airy, meme-focused concept. 
- Accent color: Electric purple (#A020F0) to make key information 'pop' like links and calls to action.
- Font: 'Space Grotesk' sans-serif for headlines, and 'Inter' for body text to convey the blend of technology with a 'doggie' meme.
- Use dog-themed icons and meme-style graphics throughout the site.
- A single-page design with clear sections for 'About,' 'Token Details,' and 'Social Links.'
- Add small, fun animations like a dog tail wagging on scroll.
